# ForgeFit Online 💪

**Personalized Online Fitness Coaching by Kevin**

A modern, high-converting single-page website for ForgeFit Online — an online fitness coaching program.

## 🌐 Live Demo

**Production URL:** https://fitforge-online.vercel.app (deployed via Vercel)

## ✨ Features

- Beautiful dark modern design with emerald accents
- Fully responsive (mobile-first)
- Interactive elements: FAQ accordion, trial booking modal, smooth scrolling
- Multiple program cards with details
- Transparent pricing with popular plan highlighted
- Real testimonials (including Kenyan clients)
- 7-day free trial flow with form
- WhatsApp integration for direct coach contact
- Keyboard shortcuts & accessibility friendly

## 🛠 Tech Stack

- Pure HTML5 + Tailwind CSS (via CDN)
- Vanilla JavaScript
- Font Awesome icons
- No build step required — works instantly

## 🚀 Deployment

### GitHub Pages
The site can be hosted directly via GitHub Pages (enable in repo settings).

### Vercel (Recommended)
1. Import the repository into [Vercel](https://vercel.com)
2. Vercel automatically detects it as a static site
3. Deploy with zero configuration
